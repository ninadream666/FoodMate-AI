## 测试方案概要

- **测试方法**：基于 ISO 9241-11 任务式可用性测试 + System Usability Scale 问卷
- **被试人数**：拟招募 5–8 名
- **被试画像**：本科生 / 研究生，无 FoodMate-AI 使用经验，每周外卖 ≥ 3 次
- **测试时长**：25–35 分钟 / 每位
- **任务数量**：7 个核心任务（注册 → 健康信息 → 浏览 → 关键词搜索 → AI 解释 → 拍照识别 → 下单）
- **数据采集**：`usability_data_collector.py` 自动计时 + 记录错误数、求助数
- **数据汇总**：`usability_aggregator.py` 自动算 SUS 总分、任务完成率、平均用时

## 测试预期目标（基于 Nielsen 行业标准）

| 指标 | 目标值 | 依据 |
|------|--------|------|
| SUS 平均分 | ≥ 68 | Bangor 2009 行业均值 |
| 任务完成率 | ≥ 90% | Nielsen "5-User Test" |
| 关键任务用时 | < 60 秒 | 移动端期望响应时间 |
| 严重可用性问题 | ≤ 2 个 | 启发式评估常用阈值 |
