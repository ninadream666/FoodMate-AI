  二、全局限流过滤器 (RateLimitFilter.java) — 新增

  在无 API Gateway 的情况下，每个服务实例自行限流：
  - 全局 QPS: 500 req/s 上限
  - 单 IP QPS: 100 req/s，防止单点刷接口
  - 并发连接数: 最多 300 个同时处理
  - 超限返回 429，健康检查/Actuator 端点豁免

  - getMyOrders: @CircuitBreaker(orderService) — 数据库故障时快速熔断
  - payOrder: @RateLimiter(createOrder) — 防止支付刷单
