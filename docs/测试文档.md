## 一、文档概述

### 1.1 项目简介

FoodMate-AI 是一个基于微服务架构的智能外卖推荐平台，集成了多模态人工智能能力，包括基于大语言模型（DeepSeek）的个性化推荐、基于计算机视觉（Gemini Vision + EfficientNet）的食物识别与营养分析、以及基于AI的动态定价。系统包含6个Java微服务（Spring Boot 3.2 / Spring Cloud 2023.0.0）和3个Python AI微服务（FastAPI），通过Eureka实现服务发现，OpenFeign实现同步通信，RabbitMQ实现异步消息传递。前端采用React Native实现跨平台移动端，React + Vite实现Web管理端。

### 1.2 测试目的

本次测试旨在对FoodMate-AI平台进行全面的质量验证，覆盖从单个函数级别的单元测试到完整用户场景的验收测试。通过多层次、多维度的测试，确保系统在功能正确性、数据一致性、安全性和性能方面满足上线要求。所有测试数据均来自实际测试运行结果，不含估算数据。
